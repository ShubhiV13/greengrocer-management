## Green Grocery Management System


### Overview

- Green Grocery Management System is a web-based application developed using Python, HTML, and CSS to help manage grocery store operations efficiently. The system provides features for product management, sales tracking, user management, and point-of-sale (POS) functionality through a simple and user-friendly interface.

### Features

- User Registration and Login
- Dashboard for store management
- Add new grocery products
- Edit and update product details
- View and manage product inventory
- Point of Sale (POS) system for billing
- Sales tracking and management
- User management functionality
- Responsive and easy-to-use interface

## Technologies Used

- Language: Python
- Framework: Flask
- Frontend: HTML, CSS
- Deployment: Render
- IDE: VS Code

### Installation/Setup

1. Install Python on your computer.
2. Clone or download this repository:
- git clone https://github.com/ShubhiV13/greengrocer-management.git
3. Navigate to the project directory:
- cd green-grocery-management
4. Install dependencies:
- pip install -r requirements.txt
5. Run the application:
- python app.py
6. Open your browser and visit:
- http://localhost:5000

### Files

### Backend
- app.py – Main Flask application.
- requirements.txt – Project dependencies.
- render.yaml – Deployment configuration for Render.

### Templates
- base.html – Base template layout.
- login.html – User login page.
- register.html – User registration page.
- dashboard.html – Admin dashboard.
- products.html – Product listing page.
- add_product.html – Add new products.
- edit_product.html – Update product details.
- sales.html – Sales management page.
- users.html – User management page.
- pos.html – Point of Sale (POS) page.

### Static Files
- style.css – Styling for the application.
